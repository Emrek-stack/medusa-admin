Find and present reusable frontend patterns only.

## Include
- Page layout patterns
- Table/list/filter patterns
- Form and modal patterns
- Theme/i18n usage
- Loading/error skeleton usage

## Exclude
- Backend endpoint design
- Server-side business logic
- Database patterns
