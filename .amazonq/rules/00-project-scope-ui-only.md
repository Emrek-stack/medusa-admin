# Project Scope (UI-Only)

This repository is a standalone admin web app.

## Allowed Work
- Frontend UI only.
- React components, pages, routes, providers, i18n, theme, and styling.
- Local mock data and frontend-only helper utilities.

## Disallowed Work
- Backend service implementation.
- API server setup or server contracts.
- Database schema/migrations.
- Infrastructure, deployment, and ops changes.
- Medusa backend integration work.

## Preferred Paths
- `src/components/**`
- `src/routes/**`
- `src/providers/**`
- `src/i18n/**`
- `src/lib/**` (frontend-only utilities and mocks)
